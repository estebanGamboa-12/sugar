# Atelier Noir · Tartas Couture

Experiencia de scrollytelling cinematográfica en Next.js App Router con GSAP ScrollTrigger + Lenis.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Producción

```bash
npm run build
npm run start
```

## Stack

- Next.js (App Router)
- React 18
- TailwindCSS
- GSAP + ScrollTrigger
- Lenis
